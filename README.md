# superstore-sales-dashboard
Excel Sales Performance Dashboard using Pivot Tables and Slicers
## Project Summary

This project presents an interactive Sales Performance Dashboard built in Microsoft Excel using the Superstore dataset (10,195 records).

The goal of this project was to transform raw transactional sales data into meaningful business insights using Pivot Tables, Pivot Charts, Slicers, and calculated metrics.

The dashboard provides a high-level overview of revenue, profitability, customer performance, and regional trends in a clean and interactive format.
## Business Problem

Management needs a clear and interactive way to:

- Monitor total revenue and profit performance
- Identify the most profitable regions
- Analyze category-level sales contribution
- Understand customer concentration and top contributors
- Track overall profit margins

Raw data alone does not provide actionable insight, so a dynamic dashboard was developed.
## Dataset Information

- Source: Superstore Sales Dataset
- Total Records: 10,195
- Main Sheet Used: Orders
- Key Fields:
  - Order Date
  - Region
  - Category
  - Segment
  - Sales
  - Profit
  - Customer Name
  - Quantity
  - Discount

## Dashboard Features

- Sales & Profit by Region (Combo Chart)
- Sales & Profit by Category (Horizontal Bar Chart)
- Top 10 Customers by Sales
- Interactive Filters using Slicers (Region, Category, Segment, Order Date)
## Key Insights

- The West region recorded both the highest sales and highest profit, making it the strongest performing region overall.
- The South region generated the lowest sales; however, it did not record the lowest profit, indicating relatively better cost efficiency compared to some regions.
- The Central region recorded the lowest profit despite not having the lowest sales, suggesting potential margin or cost management issues.
- Technology was the best-performing category, contributing the highest sales and highest profit.
- Office Supplies generated the lowest sales among all categories.
- Furniture recorded the lowest profit, indicating possible pricing, discounting, or cost structure concerns.
- Sean Miller was the highest revenue-generating customer with total sales of $25,043.05.
